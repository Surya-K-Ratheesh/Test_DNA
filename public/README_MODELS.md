Place face-api.js model files in this folder at runtime.

Required files (download from the face-api.js models repository):
- face_landmark_68_model-weights_manifest.json
- face_landmark_68_model-shard1
- tiny_face_detector_model-weights_manifest.json
- tiny_face_detector_model-shard1

Recommended path: `public/models/` so the app can load them from `/models/`.

You can get prebuilt models from: https://github.com/justadudewhohacks/face-api.js/tree/master/weights
